# Журнал инцидентов (learning ratchet)

Агент не учится между сессиями — переживают только файлы и механические гейты. Поэтому значимые ошибки фиксируются здесь: повторная ошибка одного класса должна приводить к механизму (хук, валидатор), а не к очередному напоминанию. Этот файл читается в начале каждой сессии (навык `sync`).
