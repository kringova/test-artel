# Открытые запросы пользователя

Многошаговые и отложенные просьбы пользователя, которые нельзя терять. Заноси сюда незакрытым пунктом сразу, как только пользователь просит что-то многошаговое или на потом; отмечай `[x]` по выполнении. Навык `sync` показывает незакрытые в начале сессии.

Формат: `- [ ] <запрос> · <дата> · <где/статус>`
