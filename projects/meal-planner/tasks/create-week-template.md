## Что нужно сделать
Создать простой повторно используемый формат для меню на семь дней и связанного списка покупок по категориям.

## Почему важно
Единый шаблон сокращает время следующего планирования и не даёт пропустить день или категорию продуктов.

## Критерии готовности (DoD)
- [ ] В шаблоне есть семь дней и выбранные приёмы пищи
- [ ] Для каждого блюда можно указать основные ингредиенты
- [ ] Список покупок сгруппирован по понятным категориям
- [ ] Шаблон пригоден для повторного заполнения на следующую неделю
